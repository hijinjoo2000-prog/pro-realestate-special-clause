# PRO 재개발 특약 생성기 - 웹앱 버전

부동산 재개발 특약 문구를 자동 생성하는 Streamlit 웹 앱입니다.

## 🚀 Streamlit Cloud 배포 방법

1. **GitHub에 이 폴더 내용 업로드**
   - `app.py`, `requirements.txt` 두 파일을 GitHub 새 저장소에 업로드

2. **Streamlit Cloud에서 배포**
   - https://share.streamlit.io 접속 (Google/GitHub 계정으로 로그인)
   - "New app" 클릭
   - GitHub 저장소 선택 → `app.py` 선택 → Deploy!

3. **영구 URL 발급**
   - 배포 완료 후 `https://xxx.streamlit.app` 형태의 영구 URL 제공
   - 이 URL을 카카오톡에 저장해두면 어디서든 폰/PC로 접속 가능

## 📱 모바일 사용법

1. 브라우저에서 URL 접속
2. 입력 폼에 계약 정보 입력 → 실시간으로 특약 자동 생성
3. 결과창 텍스트 길게 눌러 전체선택 → 복사 → 카카오톡 붙여넣기
4. 또는 "텍스트 파일로 저장" 버튼으로 파일 다운로드
